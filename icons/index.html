# 使用方法：

  • 全部导入

    1. 先导入：
    import HaIcons from 'haru-ui/dist/ha-icons' // 全部导入
    1. 然后在需要使用图标的地方应用HaIcons['图标名称'], 例如在通知组件中：





     


    this.$haru.notify(this, {
  msg: '这是一个haru UI的通知',
  duration: 300000,
  location: 'topRight',
  bgColor: 'bgc-lightBlue',
  icon: HaIcons['address']
})

  • 按需导入单个图标

    import 图标名 from 'haru-ui/lib/icons/TmallElves/图标名.svg'
